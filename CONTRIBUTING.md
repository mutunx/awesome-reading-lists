# Contributing

Awesome Reading Lists accepts suggestions that improve topic quality, metadata accuracy, and reading order.

## Book Suggestions

Each suggested book should include:

- Title
- Author
- ISBN, if available
- Target topic
- Short reason for inclusion
- Suggested reading position, if the order matters

## Quality Criteria

Recommended books should be useful for long-term reading, not only short-term trends. Prefer books with clear topic fit, stable editions, broad reader value, or strong cultural and educational relevance.

## Metadata Rules

- Use the exact title and author shown by the edition you are recommending.
- Include ISBN when possible.
- Do not add download links, pirated file links, or copyrighted book text.
- Keep summaries original and concise.

## Topic Changes

When adding a new topic, include:

- A topic summary
- Who the list is for
- Recommended reading order
- At least 10 curated books
- A practical FAQ
- Related guides
- A matching entry in `topics.json`
